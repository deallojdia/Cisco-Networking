# DEA High School Network Project
Academic Project at University
Description:
This project presents the design and implementation of a computer network for "DEA" High School, supporting over 300 devices across three floors. The network ensures security, scalability, and efficient management for both administrative and educational activities.

Technologies Used:
Cisco Packet Tracer

Key Features:
Segmented network with VLANs for offices, labs, and classrooms
DHCP server and static IPs for critical devices
Access Control Lists for traffic and security management
Wi-Fi coverage on all floors
Scalable and maintainable network infrastructure
Testing and monitoring for reliable performance

What I Learned:
Network planning and requirements analysis
Physical and logical design with VLANs and VLSM
Cost analysis and infrastructure budgeting
Network testing, monitoring, and maintenance
